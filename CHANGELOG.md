# Changelog

Все важные изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-10-31

### Добавлено
- Автоматическая установка 3X-UI панели
- Настройка протокола VLESS с Reality
- Автоматическая генерация криптографических ключей (X25519, UUID, ShortID)
- Системная оптимизация для VPN-трафика:
  - Активация TCP BBR congestion control
  - Включение TCP Fast Open
  - Увеличение TCP буферов до 128 МБ
  - Оптимизация file descriptors (до 1,000,000)
  - Настройка network backlog и очередей
  - Оптимизация параметров памяти (swap, кэширование)
- Автоматическая настройка UFW firewall
- Защита от SYN flood и IP spoofing
- Генерация подробного конфигурационного файла
- Интерактивный режим установки
- Поддержка Ubuntu 20.04+ и Debian 10+
- Цветной вывод для удобства использования
- Проверка версии ядра для BBR
- Автоматическая загрузка модуля tcp_bbr

### Безопасность
- Генерация уникальных криптографических ключей
- Настройка firewall с минимально необходимыми открытыми портами
- Защита от сетевых атак на уровне ядра
